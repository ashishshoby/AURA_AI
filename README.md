# AURA AI

AURA AI is a Python-powered desktop assistant designed to automate everyday computer tasks through voice commands and intelligent application management.

## Features

- Voice command recognition
- Text-to-Speech responses
- Open installed desktop applications
- Close running applications
- Intent-based command processing
- Application scanning and management
- System information retrieval
- Modular and extensible architecture

## Technologies Used

- Python 3
- SpeechRecognition
- Edge-TTS
- PyAudio
- psutil
- pyttsx3
- Windows APIs

## Project Structure

```
AURA-AI/
│
├── src/                # Core modules
├── data/               # Configuration and resources
├── docs/               # Documentation
├── main.py             # Application entry point
├── requirements.txt    # Project dependencies
└── README.md
```

## How It Works

AURA AI listens for user voice commands, processes the intent, and executes corresponding actions such as:

- Opening applications
- Closing applications
- Providing spoken responses
- Managing system tasks
- Retrieving information

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/AURA-AI.git
cd AURA-AI
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Linux:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## Run AURA AI

```bash
python main.py
```
